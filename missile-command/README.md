# Missile Command

A small, free clone of the classic 1980s arcade game. No installation
required — it runs in your web browser (Chrome, Edge, Firefox, whatever
you have on Windows).

## How to play it

1. Copy this whole folder (`missile-command`) anywhere on your PC — the
   Desktop is fine.
2. Double-click **`index.html`**. It will open in your web browser and
   the game will be ready to play.
   - If that doesn't work for any reason, try double-clicking
     **`Play Missile Command.bat`** instead — it does the same thing.
   - If Windows shows a blue "Windows protected your PC" warning the
     first time you run the `.bat` file, click **More info**, then
     **Run anyway**. This happens for any script you download from the
     internet — it's just Windows being cautious, not a sign anything
     is wrong. You can avoid it entirely by using `index.html` instead.
3. Pick a difficulty (Easy is a good place to start) and click
   **START GAME**.

## How to play the game itself

- Enemy missiles fall from the top of the screen toward your **cities**
  (blue) and your **missile batteries** (green triangles at the
  bottom).
- **Click anywhere on the screen** to fire an interceptor from your
  nearest battery. It explodes where you clicked — anything caught in
  the blast is destroyed.
- Each battery has limited ammo per wave (shown as a number underneath
  it), but it refills at the start of every new wave.
- Protect your cities as long as you can. The game ends when all of
  them are destroyed.
- Score bonus points for every city and battery still standing at the
  end of a wave.

### Controls

| Key / action        | What it does                  |
|----------------------|--------------------------------|
| Click                | Fire an interceptor             |
| `P` or `Esc`          | Pause / resume                  |
| `M`                   | Mute / unmute sound              |

Your high score is saved automatically in the browser, so it'll still
be there next time you play.

## Nothing to install, nothing to worry about

This is a single file (`index.html`) — everything (graphics, sound,
game logic) is built in, with no downloads, accounts, ads, or internet
connection required to play. It won't install anything on your
computer or change any settings.

Have fun!
